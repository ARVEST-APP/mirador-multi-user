## What you need to do 

- Clone this repo : `git clone git@github.com:SCENE-CE/mirador-multi-user.git`
- `nvm use`
- `cd backend`
- `npm install`
- `cd ../frontend-TS`
- `npm install`
- `cd ..`
- `docker-compose up --build`

Now you can access :
- frontend to `localhost:4000`
- backend to `localhost:3000`
- DB to `localhost:3306`

## For Production
